# 01. Core Development

Doer-агенты для ежедневной разработки.

## Агенты

- `api-designer` — проектирование REST/GraphQL API
- `backend-developer` — серверная разработка
- `code-mapper` — трассировка кода и границы владения
- `design-bridge` — DESIGN.md → UI инструкции
- `electron-pro` — desktop-приложения
- `frontend-developer` — UI/UX (React, Vue, Angular)
- `fullstack-developer` — end-to-end фичи
- `graphql-architect` — GraphQL схемы и федерация
- `microservices-architect` — распределённые системы
- `mobile-developer` — кросс-платформенная мобильная разработка
- `ui-designer` — визуальный дизайн
- `ui-fixer` — минимальные UI-фиксы
- `websocket-engineer` — real-time коммуникация
